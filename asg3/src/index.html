Controls

W = Forward | S = Backward | A = Move Left | D = Move Right

Q = Rotate Left | E = Rotate Right

Mouse Drag = Rotate Camera

F = Add Block | G = Remove Block

Features

32x32 procedurally generated world

Variable wall heights

Multiple textures

Editable voxel blocks

Perspective camera

Skybox and textured cubes using UV mapping
